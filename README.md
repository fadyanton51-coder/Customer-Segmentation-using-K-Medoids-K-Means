# Customer Segmentation using Clustering

## 📌 Overview

This project applies clustering techniques to segment customers based on their insurance data. The goal is to identify meaningful groups using unsupervised learning.

## 🧠 Techniques Used

* K-Medoids Clustering
* K-Means Clustering
* Silhouette Score for evaluation
* Elbow Method for optimal K selection
* Data preprocessing and scaling

## ⚙️ Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## 📊 Process

1. Data Cleaning (removing duplicates and missing values)
2. Feature Selection (numerical features)
3. Data Scaling using StandardScaler
4. Finding optimal number of clusters using Silhouette Score
5. Applying K-Medoids clustering
6. Comparing results with K-Means using Elbow Method

## 📈 Results

* Optimal number of clusters was determined using Silhouette Score
* Customers were segmented into distinct groups
* Each cluster shows different behavioral patterns
