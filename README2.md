# Customer Segmentation using Hierarchical Clustering

## 📌 Overview

This project applies hierarchical clustering techniques to segment customers based on insurance data. It helps in understanding data structure using dendrograms and clustering analysis.

---

## 🧠 Techniques Used

- Agglomerative Hierarchical Clustering  
- Dendrogram Visualization  
- Silhouette Score for evaluation  
- Custom Elbow Method (WCSS)  
- Data preprocessing and scaling  

---

## ⚙️ Tools & Libraries

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn  
- SciPy  

---

## 📊 Process

1. Load dataset  
2. Encode categorical variables using one-hot encoding  
3. Standardize features using StandardScaler  
4. Build dendrograms for different linkage methods:
   - Ward  
   - Complete  
   - Average  
5. Apply Agglomerative Clustering  
6. Evaluate using Silhouette Score  
7. Find optimal number of clusters  
8. Visualize final clusters  

---

## 📈 Results

- Dendrogram helped understand cluster structure  
- Best number of clusters selected using evaluation metrics  
- Customers grouped into meaningful segments  
- Clear separation observed in feature space
